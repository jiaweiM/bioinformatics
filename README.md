## 工具

- [BioPython]
- [scikit-bio]
- [biocode](https://github.com/jorvis/biocode)
- [DEAP](https://deap.readthedocs.io/en/master/)

进化计算框架，用于快速原型设计和测试。

- [Nilearn](https://nilearn.github.io/stable/index.html)

神经成像统计分析 Python 包。

- [PsychoPy](https://www.psychopy.org/)

用于行为科学（神经学、心理学、心理物理学、语言学等）分析测试。
