## 简介

`cp` 命令为 copy 缩写，用于复制文件或目录。

```sh
cp [参数] 源文件 目标文件
```

|参数|说明|
|---|---|
|-f|若目标文件已存在，则会直接覆盖原文件|
|-i|若目标文件已存在，则会询问是否覆盖|
|-p|保留源文件或目录的所有属性|
|-r|递归复制文件和目录|
|-d|当复制符号连接时，把目标文件或目录也建立为符号连接，并指向与源文件或目录连接的原始文件或目录|
|-l|对源文件建立硬连接，而非复制文件|
|-s|对源文件建立符号连接，而非复制文件|
|-b|覆盖已存在的文件目标前将目标文件备份|
|-v|详细显示cp命令执行的操作过程|
|-a|等价于“pdr”选项|

## 实例

- 在当前工作目录将文件复制一份，定义新名称

```sh
cp anaconda-ks.cfg kickstart.cfg
```

- 将文件复制到 `/etc` 目录，并覆盖已有文件

```sh
cp -f ks.cfg /etc
```
