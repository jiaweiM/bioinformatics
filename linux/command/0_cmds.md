# Linux 命令大全

命令|功能
---|---
[cal](cal.md)|显示当前月份的日历
[cd](cd.md)|切换工作目录
[date](date.md)|显示当前时间和日期
[df](df.md)|磁盘可用空间
[exit](exit.md)|退出终端
[file](file.md)|查看文件类型
[free](free.md)|查看内容
[less](less.md)|查看文本文件
[ls](ls.md)|查看目录内容
[pwd](pwd.md)|输出工作目录
