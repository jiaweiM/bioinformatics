# PyMOL 命令

## select

https://pymolwiki.org/index.php/Select

`select` 从选择的原子创建命名选择。

```sh
select name [, selection [, enable [, quiet [, merge [, state ]]]]]
```

- `name`

选择对象的名称，默认为 `sele`。

- `selection`
